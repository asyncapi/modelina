# Kotlin use custom type mappings

A basic example on how to use custom type mappings for Kotlin, such as mapping `number` to `BigDecimal` and `date-time` strings to `Instant`.

## How to run this example

Run this example using:

```sh
npm i && npm run start
```

If you are on Windows, use the `start:windows` script instead:

```sh
npm i && npm run start:windows
```
